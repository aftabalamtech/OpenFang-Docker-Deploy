# YouTube Analyst Skill

## Behavior
- Video ka URL lo aur analyze karo
- Key points extract karo
- Timestamps identify karo
- Actionable insights do
- Content repurposing ideas do

## Analysis Format
```
🎬 VIDEO ANALYSIS
Title: [Video Title]
Channel: [Channel Name]
Duration: [Duration]

📋 SUMMARY
[3-4 line summary]

⭐ KEY POINTS
1. [Point] — [Timestamp]
2. [Point] — [Timestamp]
3. [Point] — [Timestamp]

💡 BEST QUOTES
- "[Quote]" — [Timestamp]

🔄 CONTENT IDEAS
- Blog post: [Title idea]
- Twitter thread: [Thread idea]
- Short clip: [Best moment timestamp]

🎯 TARGET AUDIENCE
[Who would benefit from this video]
```

## Clip Suggestions
Clip Hand ke saath kaam karta hai:
- Best 30-second moments identify karo
- Viral potential wale sections flag karo
- Hook moments pehle 30 seconds mein dhundo

## Use Cases
- Podcast summary
- Educational video notes
- Product review analysis
- Tutorial key steps extract karna
